Voxel Struct Reference

O struct Voxel reuni as caracteristicas que definem a menor unidade que compõe o desenho 3D,sendo representado por um cubo de lado 1, possuindo os atributos de cor, sendo armazenado no padrão RGB, representados pelas variáveis 'r','g','b', com cada intensidade definida no intervalo aberto de 0 à 1, tendo também o atributo 'a' que define o nivel de transparência do voxel, se ele vai ser translúcido ou opaco, por fim o último atributo é o 'isOn' que define se o voxel vai aparecer no desenho ou não. Detailed Description

O struct Voxel reuni as caracteristicas que definem a menor unidade que compõe o desenho 3D,sendo representado por um cubo de lado 1, possuindo os atributos de cor, sendo armazenado no padrão RGB, representados pelas variáveis 'r','g','b', com cada intensidade definida no intervalo aberto de 0 à 1, tendo também o atributo 'a' que define o nivel de transparência do voxel, se ele vai ser translúcido ou opaco, por fim o último atributo é o 'isOn' que define se o voxel vai aparecer no desenho ou não.
